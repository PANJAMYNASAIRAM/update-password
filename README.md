##Project Description

This project is a simple Password Update Form built using JavaScript that validates user input in real-time. It ensures that the user enters a new password correctly and confirms it by matching both fields before submitting the form.

The application improves user experience by providing instant feedback and preventing invalid form submissions.

##Features

🔐 New Password Validation
Displays an error message if the field is empty 

✅ Confirm Password Validation
Checks whether both passwords match
Shows an error message if they do not match 

⚡ Real-time Validation
Validation triggers when the user leaves (blur event) input fields 

🚫 Prevents Form Submission
Stops submission if validation fails 

📢 User-friendly Error Messages
Clear and simple messages for better UX

##Technologies Used

HTML5 – Structure of the form
CSS3 – Styling and layout
JavaScript (ES6) – Form validation logic and DOM manipulation 

##Installation Steps

1. Clone the repository git clone https://github.com/PANJAMYNASAIRAM/update-password.git
2. Go to project folder "cd update-password"
3. Install dependencies "npm install"
4. Start the project "npm start"

##How It Works
User enters a new password
User confirms the password
System checks:
If fields are empty
If passwords match
Displays error messages if validation fails
Prevents submission until all validations pass
